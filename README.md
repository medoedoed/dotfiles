### Specs

|----------------------|---------------------------|

| **WM:**           | sway + waybar     | 
| **Editor:**       | nvim                      | 
| **Terminal:** | foot                        |
| **Launcher:** | wofi                       | 
|**Distro:** | fedora 40 |


------

### Installing

To install all configs to ~/.config just run ./install.sh.

-----

### Setup

 You can setup installing and choose that dotfiles you want to copy with comment/uncomment choosen configs in ./setup.sh.
